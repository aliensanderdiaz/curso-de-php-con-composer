Sección 10 de 19 - Composer

Creamos composer.json

            
        {
            "name": "aliensanderdiaz/text",
            "autoload": {
                "psr-4": {
                    "Text\\": "src/"
                },
                "files": [
                    "src/helpers.php"
                ]
            }
        }

creo la carpeta src
creo el archivo src/helpers.php
creo la clase Format.php 

            
        <?php

        namespace Text;
        
        class Format
        {
            public static function upperText($value)
            {
                return strtoupper($value);
            }
        }

ejecuto en la terminal 

            
        > composer dump

Esto me crea la carpeta vendor

creo el archivo index.php 

            
        <?php

        require __DIR__ . '/vendor/autoload.php';
        
        // echo Text\Format::upperText("camila");
        echo upper('Cami');

La linea comentada es como se haria hasta el momento, pero es algo demasiado largo para algo tan sencillo.
entonces para ello usaremos el archivo helpers.php 

            
        <?php

        function upper($value) {
            return Text\Format::upperText($value);
        }

Sección 11 de 19 - Revisión del proyecto

Agregamos otro metodo a nuestra clase

            
      <?php

      namespace Text;
      
      class Format
      {
          public static function upperText($value)
          {
              return strtoupper($value);
          }
          public static function lowerText($value)
          {
              return strtolower($value);
          }
      }

Creamos el respectivo helper

NOTA: Agregamos una comprobación a cada función para evitar duplicidad de funciones (duplicar funciones es un error fatal)

            
      <?php

      if (!function_exists('upper')) {
          function upper($value)
          {
              return Text\Format::upperText($value);
          }
      }
      
      if (!function_exists('lower')) {
          function lower($value)
          {
              return Text\Format::lowerText($value);
          }
      }

Lo utilizamos en el index

            
      <?php

      require __DIR__ . '/vendor/autoload.php';
      
      // echo Text\Format::upperText("camila");
      echo upper('Cami');
      echo '<br>';
      echo lower('Cami');
